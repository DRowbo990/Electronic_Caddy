GPS:
PC5 - RX

LCD:
PB0-3 data
PC10 E
PC11 RW
PC12 RS

Keypad:

PB4-7 Rows: Open-drain, output
PB8-11 Cols: Input, external pull-up 2.2k and 3.3V
NOTE: Maybe use internal pull-up instead

Motor:
C1,2 = input pins for motor encoder
C2 = White encoder line
C3 = Yellow encoder line

C3 = output pin for motor spinning
C3 = Black wire, first on driver
C0 = Orange wire, second on driver
C0 = other output for direction control
